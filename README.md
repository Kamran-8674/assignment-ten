## FlavorTrail

## Features:
- In this website user can give reviews of the foods they have tasted. Also user can see their own reviews and their favorite reviews.
- learn about great foods that user has never heard of.
- learn about some amazing local chefs
- authentication is also implemented 




 ## Technology used for this project
- React.js
- Node.js
- JWT
- DaisyUI
- Firebase
- React Icons
- React Router
- React Toastify
- Sweetalert
- Swiper
- TailwindCSS
